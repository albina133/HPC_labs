# Фильтрация шума SALT AND PEPPER на gpu

## Цель работы

Реализация фильтрации изображения с шумом типа salt and pepper с использованием **cuda**. 
Необходимо было:
- взять входное изображение в формате **bmp**;
- применить **9-точечный медианный фильтр**;
- выполнить обработку на **gpu**;
- использовать **texture memory**;
- сохранить результирующее изображение;
- вывести время обработки на gpu.

## Выполнение задания

В программе выполнены следующие шаги:
1. загружается исходное изображение `thai.bmp`;
2. изображение читается вручную через функции работы с bmp;
3. исходное изображение переводится в оттенки серого;
4. к grayscale-изображению программно добавляется шум типа **salt and pepper**;
5. зашумлённое изображение копируется в память gpu;
6. входные данные привязываются к **texture memory**;
7. для каждого пикселя применяется **медианный фильтр 3x3**;
8. результат копируется обратно на cpu;
9. итоговое изображение сохраняется в формате bmp;
10. дополнительно выводится время обработки на gpu.

## Реализация в коде

Используются следующие основные части:

### Чтение bmp-файла

Функция `readBmp()` считывает изображение формата bmp и преобразует его во внутренний rgb-буфер.

### Сохранение изображений

Для сохранения результатов используются функции:
- `writeBmpRgb()` — сохраняет rgb-изображение;
- `writeBmpGray()` — сохраняет grayscale-изображение.

Программа сохраняет не только итоговый результат, но и промежуточные этапы обработки.

### Перевод в grayscale

Функция `rgbToGrayLikeNotebook()` переводит исходное цветное изображение в оттенки серого. Это необходимо, потому что фильтрация выполняется для одноканального изображения.

### Добавление шума

Функция `addSaltAndPepperNoise()` добавляет случайный импульсный шум. Часть пикселей заменяется на `0` или `255`, то есть на чёрные и белые точки.

### Фильтрация на gpu

Основная обработка выполняется в kernel-функции:

```cpp
__global__ void medianFilterTexture(...)
```

В этой функции:
- для каждого пикселя берётся окно **3x3**;
- соседние значения читаются через **texture memory**;
- значения сортируются простым методом `bubble sort`;
- в выходное изображение записывается центральный элемент отсортированного массива, то есть медиана.

Базовая реализация **9-точечного медианного фильтра**.

### Измерение времени

Время работы gpu-части измеряется с помощью `cudaEvent_t`, то есть отдельно для запуска и выполнения kernel-функции.

## Работа программы

После запуска программа выполняет обработку автоматически. На вход подаётся файл:

```text
thai.bmp
```

После выполнения создаются следующие файлы:

- `step_rgb_original.bmp` - исходное изображение;
- `gray_image.bmp` - изображение после перевода в grayscale;
- `input_image.bmp` - изображение после добавления шума;
- `output_image.bmp` - результат после медианной фильтрации.

## Выводы

В ходе лабораторной работы была реализована программа фильтрации шума типа salt and pepper на gpu с использованием cuda. В процессе выполнения работы был изучен принцип применения медианного фильтра 3x3, использована texture memory для чтения пикселей изображения, а также реализована обработка изображения в формате bmp.
